### 学习目标

- 理解自然语言处理的基本流程
- 掌握中英文分词的方法与工具
- 理解词向量的概念与构建方式
- 掌握 TF-IDF 算法的原理与实现

### 核心知识点

#### 1. 文本预处理流程

```
原始文本 → 分词 → 去停用词 → 词性标注 → 命名实体识别 → 向量化
```

#### 2. 分词方法

| 方法 | 原理 | 适用场景 |
|------|------|----------|
| 规则分词 | 词典+最大匹配 | 词典丰富的领域 |
| 统计分词 | HMM/CRF 模型 | 歧义较多的场景 |
| 深度学习分词 | BiLSTM-CRF | 通用场景，效果最好 |

#### 3. 词向量表示

- **One-Hot**：维度爆炸，稀疏，无法表达语义相似性
- **Word2Vec**：分布式表示，稠密向量，通过上下文学习语义
- **Embedding**：将离散词映射到连续向量空间

#### 4. TF-IDF 公式

```
TF-IDF = TF(t,d) × IDF(t)
TF(t,d) = 词t在文档d中出现的次数 / 文档d的总词数
IDF(t) = log(文档总数 / 包含词t的文档数 + 1)
```

    words = [w for w in words if w.strip() and w not in stopwords and len(w) > 1]
    return words

    """计算词频"""

## L14: Transformer与注意力机制

### 学习目标

- 理解自注意力机制（Self-Attention）的原理
- 掌握多头注意力（Multi-Head Attention）的实现
- 理解 Transformer 架构的核心组件
- 能够从零实现一个简化的 Transformer 编码器

### 核心知识点

#### 1. 注意力机制公式

```
Attention(Q, K, V) = softmax(QK^T / √d_k) × V
```

- Q（Query）：查询向量，代表当前词
- K（Key）：键向量，用于匹配
- V（Value）：值向量，包含实际信息
- √d_k：缩放因子，防止点积过大导致梯度消失

#### 2. 多头注意力

```
MultiHead(Q, K, V) = Concat(head_1, ..., head_h) × W^O
head_i = Attention(Q × W_i^Q, K × W_i^K, V × W_i^V)
```

- 将 Q/K/V 分别投影到 h 个子空间
- 每个头学习不同的注意力模式
- 捕获不同类型的依赖关系

#### 3. Transformer 架构

```
输入 → 词嵌入 + 位置编码 → [编码器层 × N] → 输出
                          ↓
                    自注意力 + 残差
                          ↓
                    前馈神经网络 + 残差
```

#### 4. 位置编码

```python
# 三角函数位置编码
PE(pos, 2i)   = sin(pos / 10000^(2i/d_model))
PE(pos, 2i+1) = cos(pos / 10000^(2i/d_model))
```

"""

import torch
import torch.nn as nn
import math

# ============================================================
# 示例1：手写 Self-Attention
# ============================================================
class SelfAttention(nn.Module):
    """自注意力机制实现"""

    def __init__(self, embed_size, heads):
        super().__init__()
        self.embed_size = embed_size
        self.heads = heads
        self.head_dim = embed_size // heads

        assert self.head_dim * heads == embed_size, "embed_size must be divisible by heads"

        self.W_q = nn.Linear(embed_size, embed_size)
        self.W_k = nn.Linear(embed_size, embed_size)
        self.W_v = nn.Linear(embed_size, embed_size)
        self.W_o = nn.Linear(embed_size, embed_size)

    def forward(self, x, mask=None):
        batch_size, seq_len, embed_size = x.shape

        # 线性变换得到 Q, K, V
        Q = self.W_q(x)
        K = self.W_k(x)
        V = self.W_v(x)

        # 分头
        Q = Q.view(batch_size, seq_len, self.heads, self.head_dim).transpose(1, 2)
        K = K.view(batch_size, seq_len, self.heads, self.head_dim).transpose(1, 2)
        V = V.view(batch_size, seq_len, self.heads, self.head_dim).transpose(1, 2)

        # 计算注意力分数
        scores = torch.matmul(Q, K.transpose(-1, -2)) / math.sqrt(self.head_dim)

        # 应用 mask（如果提供）
        if mask is not None:
            scores = scores.masked_fill(mask == 0, float('-inf'))

        # softmax 得到注意力权重
        attention_weights = torch.softmax(scores, dim=-1)

        # 加权求和
        context = torch.matmul(attention_weights, V)

        # 合并多头并通过输出层
        context = context.transpose(1, 2).contiguous().view(batch_size, seq_len, embed_size)
        output = self.W_o(context)

        return output, attention_weights

def visualize_attention(text, model, tokenizer, layer_idx=0, head_idx=0):
    """可视化特定层和头的注意力权重"""
    inputs = tokenizer(text, return_tensors="pt")

    with torch.no_grad():
        outputs = model(**inputs, output_attentions=True)

    # 获取注意力权重
    attentions = outputs.attentions[layer_idx]  # shape: [batch, heads, seq, seq]

    # 提取指定头的注意力
    attention = attentions[0, head_idx].numpy()

    # 获取 token 列表

    print(f"Tokens: {tokens}")

    # 打印注意力矩阵
    print("\n注意力矩阵（部分）:")
    for i, token in enumerate(tokens[:8]):
        row = attention[i][:8]
        weights = " ".join([f"{w:.3f}" for w in row])

# 示例4：完整的 Transformer Encoder 层
# ============================================================
class TransformerEncoderLayer(nn.Module):
    """Transformer 编码器层"""

    def __init__(self, embed_size, heads, ff_dim, dropout=0.1):
        super().__init__()
        self.attention = SelfAttention(embed_size, heads)
        self.norm1 = nn.LayerNorm(embed_size)
        self.norm2 = nn.LayerNorm(embed_size)

        self.ffn = nn.Sequential(
            nn.Linear(embed_size, ff_dim),
            nn.ReLU(),
            nn.Linear(ff_dim, embed_size)
        )
        self.dropout = nn.Dropout(dropout)

    def forward(self, x, mask=None):
        # 自注意力 + 残差连接
        attn_out, _ = self.attention(x, mask)
        x = self.norm1(x + self.dropout(attn_out))

        # 前馈网络 + 残差连接
        ffn_out = self.ffn(x)
        x = self.norm2(x + self.dropout(ffn_out))

        return x

# 测试

### 核心知识点

#### 1. 预训练语言模型对比

| 模型 | 架构 | 训练目标 | 适用任务 |
|------|------|----------|----------|
| Word2Vec | 浅层网络 | Skip-gram/CBOW | 词向量获取 |
| ELMo | BiLSTM | 语言模型 | 序列标注 |
| BERT | Transformer Encoder | Masked LM + Next Sentence | 理解任务 |
| GPT | Transformer Decoder | 语言模型 | 生成任务 |
| GPT-2/3 | Transformer Decoder | 语言模型 | 零样本生成 |

#### 2. BERT 训练目标

- **Masked Language Model (MLM)**：遮盖 15% 的 token，预测被遮盖的词
- **Next Sentence Prediction (NSP)**：判断句子对是否为连续句子

#### 3. GPT 系列

- **单向语言模型**：仅使用左侧上下文
- **零样本/少样本学习**：通过 prompt 完成各类任务

#### 4. Word2Vec 训练方法

- **Skip-gram**：用中心词预测上下文词
- **CBOW**：用上下文词预测中心词

def bert_fill_mask():

from sklearn.metrics.pairwise import cosine_similarity
import numpy as np

def bert_sentence_similarity():
    tokenizer = BertTokenizer.from_pretrained("bert-base-chinese")
    model = BertModel.from_pretrained("bert-base-chinese")

    sentences = [
        "我喜欢吃苹果",
        "我爱吃水果",
        "今天天气很好",
        "温度升高了"

    # 获取每个句子的 [CLS] 向量

            outputs = model(**inputs)

    print(f"  {' '.join([s[:6] for s in sentences])}")
    for i, row in enumerate(similarity):
        print(f"{sentences[i][:6]}: {' '.join([f'{x:.2f}' for x in row])}")

def gpt2_text_generation():
    tokenizer = GPT2Tokenizer.from_pretrained("gpt2")
    model = GPT2Model.from_pretrained("gpt2")

    # 设置 pad token
    tokenizer.pad_token = tokenizer.eos_token

    prompt = "Artificial intelligence is transforming"
    inputs = tokenizer(prompt, return_tensors="pt")

    with torch.no_grad():
        outputs = model(**inputs)
        # 取最后一个 token 的 hidden state 预测下一个词
        logits = outputs.last_hidden_state[0, -1]

    # 简化：取 top-k 预测
    probs = torch.softmax(logits, dim=-1)
    top_probs, top_indices = torch.topk(probs, 5)

    print(f"\n提示: {prompt}")
    print("Top 5 预测的下一个词:")
    for prob, idx in zip(top_probs, top_indices):
        word = tokenizer.decode([idx.item()])
        print(f"  {word}: {prob.item():.4f}")

    print(model.wv["深度"][:10])

    print("\n'深度学习' 的相似词:")
    for word, score in model.wv.most_similar("深度学习", topn=5):

        for word, score in result:

        print(f"类比操作失败: {e}")

train_word2vec()

### 核心知识点

#### 1. 文本分类流程

```
数据收集 → 数据清洗 → 数据标注 → 数据划分（训练/验证/测试）
     ↓
特征提取（TF-IDF / 词向量 / BERT embedding）
     ↓
模型选择（CNN / RNN / Transformer）
     ↓
训练与调参 → 评估 → 部署
```

#### 2. 命名实体识别（NER）

- **任务**：识别文本中的人名、地名、机构名、日期等实体
- **常用方法**：
  - 规则匹配（正则表达式）
  - BiLSTM-CRF（深度学习）
  - BERT-CRF（当前最佳）

#### 3. 评估指标

| 指标 | 公式 | 适用场景 |
|------|------|----------|
| 准确率 (Accuracy) | 正确数 / 总数 | 类别平衡 |
| 精确率 (Precision) | TP / (TP+FP) | 关注假阳性 |
| 召回率 (Recall) | TP / (TP+FN) | 关注假阴性 |
| F1 分数 | 2×P×R / (P+R) | 综合评估 |
| AUC-ROC | 曲线下面积 | 分类排序 |

import torch
import torch.nn as nn
from transformers import (
    AutoTokenizer, AutoModelForTokenClassification,
    AutoModelForSequenceClassification,
    Trainer, TrainingArguments
)
from sklearn.metrics import accuracy_score, precision_recall_fscore_support, classification_report
import numpy as np

# ============================================================
# 项目1：中文文本分类（情感分析）
# ============================================================

# 数据准备（示例数据）
train_data = [
    ("这家餐厅的食物很好吃", "positive"),
    ("服务态度很差，等了两个小时", "negative"),
    ("环境优雅，适合约会", "positive"),
    ("价格太贵了，不推荐", "negative"),
    ("味道一般，没有特别之处", "neutral"),
    ("下次还会再来", "positive"),
    ("再也不来了，失望透顶", "negative"),
    ("性价比很高，值得推荐", "positive"),
]

test_data = [
    ("味道不错，下次还来", "positive"),
    ("太差了，完全不推荐", "negative"),
]

label2id = {"negative": 0, "neutral": 1, "positive": 2}
id2label = {v: k for k, v in label2id.items()}


def train_text_classification():
    """使用 BERT 进行文本分类"""

    # 加载中文 BERT
    model_name = "bert-base-chinese"
    tokenizer = AutoTokenizer.from_pretrained(model_name)
    model = AutoModelForSequenceClassification.from_pretrained(
        model_name,
        num_labels=3,
        id2label=id2label,
        label2id=label2id
    )

    # 数据编码
    texts = [d[0] for d in train_data]
    labels = [label2id[d[1]] for d in train_data]

    encodings = tokenizer(texts, padding=True, truncation=True, return_tensors="pt")

    # 准备数据集
    class TextDataset(torch.utils.data.Dataset):
        def __init__(self, encodings, labels):
            self.encodings = encodings
            self.labels = labels

        def __getitem__(self, idx):
            item = {key: val[idx] for key, val in self.encodings.items()}
            item["labels"] = torch.tensor(self.labels[idx])
            return item

        def __len__(self):
            return len(self.labels)

    dataset = TextDataset(encodings, labels)

    # 训练配置
    training_args = TrainingArguments(
        output_dir="./text_classification_results",
        num_train_epochs=3,
        per_device_train_batch_size=4,
        logging_steps=10,
        save_strategy="no"
    )

    # 自定义评估指标
    def compute_metrics(eval_pred):
        predictions, labels = eval_pred
        predictions = np.argmax(predictions, axis=1)
        accuracy = accuracy_score(labels, predictions)
        return {"accuracy": accuracy}

    # 训练
    trainer = Trainer(
        model=model,
        args=training_args,
        train_dataset=dataset,
        compute_metrics=compute_metrics
    )

    trainer.train()

    # 测试预测
    test_texts = [d[0] for d in test_data]
    test_labels = [label2id[d[1]] for d in test_data]

    test_encodings = tokenizer(test_texts, padding=True, truncation=True, return_tensors="pt")
    test_encodings["labels"] = torch.tensor(test_labels)

    test_dataset = TextDataset(test_encodings, test_labels)

    predictions = trainer.predict(test_dataset)
    preds = np.argmax(predictions.predictions, axis=1)

    print("\n测试集预测结果:")
    for i, (text, true_label, pred_label) in enumerate(zip(test_texts, test_labels, preds)):
        print(f"  文本: {text}")
        print(f"  真实: {id2label[true_label]}, 预测: {id2label[pred_label]}")
        print()

train_text_classification()

# 项目2：命名实体识别（NER）

def train_ner():
    """使用 BERT-CRF 进行命名实体识别"""

    # 示例数据：中文NER语料（BIO标注格式）
    # B-PER/I-PER: 人名, B-LOC/I-LOC: 地名, B-ORG/I-ORG: 机构, O: 无实体
    train_sequences = [
        ["北", "京", "是", "中", "国", "的", "首", "都"],
        ["马", "克", "思", "是", "德", "国", "的", "思", "想", "家"],
        ["阿", "里", "巴", "巴", "是", "一", "家", "电", "商", "公", "司"],
    ]

    # 对应的标签（BIO格式）
    train_tags = [
        ["B-LOC", "I-LOC", "O", "O", "O", "O", "O", "O"],
        ["B-PER", "I-PER", "O", "O", "B-LOC", "O", "O", "O", "O", "O"],
        ["B-ORG", "I-ORG", "I-ORG", "O", "O", "O", "O", "O", "O", "O", "O"],
    ]

    # 标签映射
    label_list = ["O", "B-PER", "I-PER", "B-LOC", "I-LOC", "B-ORG", "I-ORG"]
    label2id = {label: i for i, label in enumerate(label_list)}
    id2label = {i: label for label, i in label2id.items()}

    # 加载预训练模型
    model_name = "bert-base-chinese"
    tokenizer = AutoTokenizer.from_pretrained(model_name)
    model = AutoModelForTokenClassification.from_pretrained(
        model_name,
        num_labels=len(label_list),
        id2label=id2label,
        label2id=label2id
    )

    # 数据编码
    encodings = tokenizer(
        [" ".join(seq) for seq in train_sequences],
        padding=True,
        truncation=True,
        return_tensors="pt"
    )

    # 转换为标签 ID
    tag_ids = []
    for tags in train_tags:
        tag_id = [label2id[tag] for tag in tags]
        # 补齐到相同长度
        while len(tag_id) < 128:
            tag_id.append(-100)  # -100 表示忽略
        tag_ids.append(tag_id[:128])

    tag_ids = torch.tensor(tag_ids)

    class NERDataset(torch.utils.data.Dataset):
        def __init__(self, encodings, tags):
            self.encodings = encodings
            self.tags = tags

        def __getitem__(self, idx):
            item = {key: val[idx] for key, val in self.encodings.items()}
            item["labels"] = self.tags[idx]
            return item

        def __len__(self):
            return len(self.tags)

    dataset = NERDataset(encodings, tag_ids)

    # 训练
    training_args = TrainingArguments(
        output_dir="./ner_results",
        num_train_epochs=5,
        per_device_train_batch_size=2,
        logging_steps=5,
        save_strategy="no"
    )

    trainer = Trainer(
        model=model,
        args=training_args,
        train_dataset=dataset
    )

    trainer.train()

    # 预测
    test_seq = ["北", "京", "是", "中", "国", "首", "都"]
    test_input = tokenizer([" ".join(test_seq)], padding=True, truncation=True, return_tensors="pt")

    with torch.no_grad():
        outputs = model(**test_input)
        predictions = torch.argmax(outputs.logits, dim=-1)

    predicted_tags = [id2label[p.item()] for p in predictions[0]]

    print("\nNER 预测结果:")
    print(f"  句子: {''.join(test_seq)}")
    print(f"  实体: ", end="")
    current_entity = None
    for char, tag in zip(test_seq, predicted_tags):
        if tag.startswith("B-"):
            current_entity = tag[2:]
            print(f"\n  {char}", end="")
        elif tag.startswith("I-") and current_entity:
            print(char, end="")
        else:
            current_entity = None
    print()

train_ner()

def model_deployment_example():
    """模型保存、加载与部署"""

    import os
    from transformers import AutoModelForSequenceClassification, AutoTokenizer

    # 假设已有训练好的模型
    model_name = "bert-base-chinese"
    model = AutoModelForSequenceClassification.from_pretrained(model_name, num_labels=3)
    tokenizer = AutoTokenizer.from_pretrained(model_name)

    # 保存模型
    save_path = "./saved_model"
    model.save_pretrained(save_path)
    tokenizer.save_pretrained(save_path)
    print(f"模型已保存到: {save_path}")

    # 加载模型进行推理
    loaded_model = AutoModelForSequenceClassification.from_pretrained(save_path)
    loaded_tokenizer = AutoTokenizer.from_pretrained(save_path)

    # 推理函数
    def predict(text):
        inputs = loaded_tokenizer(text, return_tensors="pt", padding=True, truncation=True)
        with torch.no_grad():
            outputs = loaded_model(**inputs)
            predictions = torch.argmax(outputs.logits, dim=-1)
        return predictions.item()

    # 测试
    test_text = "这是一家很好的餐厅"
    pred = predict(test_text)
    label_map = {0: "负面", 1: "中性", 2: "正面"}
    print(f"\n输入: {test_text}")
    print(f"预测: {label_map[pred]}")

model_deployment_example()
