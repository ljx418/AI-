# AI就业导向教材开发计划

## 一、项目概述

### 1.1 项目定位

本项目定位为**国内首个AI就业导向系统教材**，面向零基础至中级水平的学员，提供从AI入门到就业的完整路径。区别于传统学术课程，本教材以"学完能找工作"为核心目标，对标大厂AI岗位技能图谱设计内容。

### 1.2 目标用户画像

| 用户类型 | 特征描述 | 学习痛点 | 核心需求 |
|---------|---------|---------|---------|
| 计算机专业学生 | CS本科/硕士在读，有编程基础 | 理论脱离实践，缺乏项目经验 | 简历项目、面试准备、就业指导 |
| 转行IT从业者 | 有开发经验，想转型AI方向 | 缺少系统学习路径，入门门槛高 | 高效学习路径、就业导向内容 |
| 初入行AI工程师 | 刚入职，想系统化提升 | 知识碎片化，跟不上技术更新 | 前沿技术、深化专项能力 |
| 传统行业从业者 | 有行业经验，想结合AI | 缺乏技术基础，难以落地 | 实用AI技能、业务场景应用 |

### 1.3 核心差异化价值

```

│                      四大核心竞争力护城河                      │
├─────────────────────────────────────────────────────────────┤

**四大护城河：**

1. **就业导向课程设计**：对标大厂AI岗位技能图谱，每个知识点精准映射到真实岗位能力要求
2. **前沿技术全覆盖**：周级更新，LLM时代核心技术一个不漏

```

│
├── 模块1: AI基础与开发环境 (Week 1-2, L1-L4)

## 三、实战项目设计

### 3.1 项目设计原则

1. **就业导向**：每个项目对标真实业务场景，而非toy project
2. **可写进简历**：项目成果可直接作为简历作品集
3. **全流程覆盖**：从需求分析到部署上线完整流程
4. **难度递进**：基础项目 → 进阶项目 → 综合项目

### 3.2 项目池详细设计

#### 第一阶段项目（模块1-2）

| 项目名称 | 业务场景 | 技术栈 | 难度 | 产出物 |
|---------|---------|--------|------|-------|
| 电商用户购买预测 | 电商平台用户行为分析 | Python/Pandas/sklearn | 基础 | 预测模型+分析报告 |
| 用户分群分析 | 会员精细化运营 | K-Means/PCA/可视化 | 基础 | 分群方案+运营建议 |

#### 第二阶段项目（模块3-4）

| 项目名称 | 业务场景 | 技术栈 | 难度 | 产出物 |
|---------|---------|--------|------|-------|
| 图像识别模型训练 | 工业质检/医学影像 | PyTorch/CNN/迁移学习 | 进阶 | 训练代码+模型权重 |
| 文本分类微调实践 | 新闻分类/情感分析 | BERT/微调技术 | 进阶 | 微调代码+API服务 |

#### 第三阶段项目（模块5-6）

| 项目名称 | 业务场景 | 技术栈 | 难度 | 产出物 |
|---------|---------|--------|------|-------|
| 智能文档问答系统 | 法律/金融文档检索 | RAG/Embedding/向量库 | 进阶 | 完整RAG系统 |
| 智能客服系统 | 电商退换货客服 | LangChain/RAG/Agent | 高级 | 可部署客服系统 |
| 自动化数据分析师 | 业务数据分析 | Agent/ReAct/工具调用 | 高级 | Agent代码+演示 |
| 代码审查Agent | 开发团队代码质量 | Code Agent/LLM API | 高级 | Agent服务+使用文档 |

## 四、学习路径设计

### 4.1 路径A：学生党（高强度）

**适用人群**：在校学生、应届生、全力转行者
**学习时间**：15-25小时/周
**完成周期**：12周（3个月）

```
Week 1-2:   [模块1] AI基础与开发环境     ████░░░░░░░ 15%
Week 3-4:   [模块2] 机器学习核心         ████████░░░ 30%
Week 5-6:   [模块3] 深度学习基础         ████████████ 50%
Week 7-8:   [模块4] NLP与大模型          ████████████ 70%
Week 9-10:  [模块5] LLM Agent与RAG      ████████████ 85%
Week 11-12: [模块6] AI前沿与综合项目     ████████████ 100%
```

**每日学习节奏（周末休）**：
- 工作日：2-3小时/天（晚间）
- 周末：4-5小时/天

### 4.2 路径B：转行党（稳妥型）

**适用人群**：在职转行者、周末学习者
**学习时间**：5-12小时/周
**完成周期**：24周（6个月）

```
Phase 1 (Week 1-6):   基础铺垫    [模块1] + [模块2]基础
Phase 2 (Week 7-12):  核心突破    [模块2]进阶 + [模块3]
Phase 3 (Week 13-18): 应用拓展    [模块4] + [模块5]
Phase 4 (Week 19-24): 综合实战    [模块6] + 项目完善
```

**每日学习节奏**：
- 工作日：1小时/天
- 周末：3-4小时/天

### 4.3 学习路径选择决策树

```
开始学习AI
    │
    ▼
你是全职还是在职？
    │
    ├─► 全职/学生 → 路径A（12周高强度）
    │
    └─► 在职转行 → 你的每周可用时间？
              │
              ├─► >15小时 → 路径A压缩版（16周）
              │
              └─► 5-12小时 → 路径B（24周稳健型）
```
