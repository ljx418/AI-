AI概述与历史

Week 1-2 · 课时 2课时

学习目标

  1. 准确陈述AI的定义，并区分「狭义AI」「通用AI」「超级AI」三个层次
  2. 按时间顺序叙述AI发展史上的三次浪潮及其核心技术标志
  3. 解释GOFAI（符号主义人工智能）的核心假设与方法论局限
  4. 说明深度学习崛起的技术背景——数据、算力、算法三要素
  5. 评估当前主流AI系统的能力边界与固有缺陷

核心知识点

人工智能三个层次

  • 强人工智能（AGI）：具备通用认知能力，能在任意领域自主学习与推理（尚未实现）
  • 弱人工智能（Narrow AI）：在特定任务上达到或超越人类水平（当前主流）
  • 超级人工智能（ASI）：在几乎所有领域远超人类智能（理论假设）

AI发展简史

年代阶段核心技术代表成果
1956-1974第一次浪潮逻辑推理、搜索树ELIZA、Shakey机器人
1980-1987第二次浪潮专家系统、规则引擎MYCIN、XCON
1990-2012统计学习SVM、随机森林垃圾邮件过滤
2012至今深度学习CNN、TransformerAlexNet、GPT系列

深度学习崛起三要素

  • 数据：互联网时代产生的海量标注数据（ImageNet百万级）
  • 算力：GPU/TPU并行计算突破，训练时间从数周压缩到数小时
  • 算法：ReLU激活函数、Dropout、Batch Norm、迁移学习

数学基础回顾

Week 1-2 · 课时 2课时

核心知识点

矩阵运算

  • 矩阵乘法：(m×n)·(n×p) = (m×p)
  • 转置：行列互换，记作Aᵀ
  • 逆矩阵：A⁻¹，满足AA⁻¹ = I

贝叶斯定理

P(A|B) = P(B|A)·P(A) / P(B)

  • P(A)：先验概率
  • P(A|B)：后验概率
  • P(B|A)：似然度

信息熵

H(X) = -∑ p(x) log₂ p(x)（单位：比特）

交叉熵：CE(p, q) = -∑ p(x) log q(x)

"""
L3-math-review.py
数学基础综合演示
安装依赖: pip install numpy matplotlib
"""

import numpy as np
import matplotlib.pyplot as plt

print("=" * 50)
print("L3 数学基础综合演示")
print("=" * 50)

# Part 1: 矩阵运算
A = np.array([[2, 1], [1, 2]])
eigenvalues, eigenvectors = np.linalg.eig(A)
print(f"\n矩阵A:\n{A}")
print(f"特征值: {eigenvalues.round(4)}")

# Part 2: 贝叶斯定理 - 医疗诊断
P_disease = 0.01
P_positive_given_disease = 0.99
P_positive_given_healthy = 0.01

# P(阳性) = P(阳性|疾病)P(疾病) + P(阳性|健康)P(健康)
P_positive = P_positive_given_disease * P_disease + P_positive_given_healthy * (1 - P_disease)

# P(疾病|阳性)
P_disease_given_positive = P_positive_given_disease * P_disease / P_positive
print(f"\n贝叶斯诊断:")
print(f"P(疾病|阳性) = {P_disease_given_positive:.4f}")
print(f"即使检测99%准确，患病率1%时，阳性结果真正患病概率仅50%!")

# Part 3: 熵的计算
def entropy(p):
    p = np.array(p)
    return -np.sum(p * np.log2(p + 1e-10))

p_coin = [0.5, 0.5]
print(f"\n均匀硬币熵: {entropy(p_coin):.4f} bits")

# Part 4: 梯度下降
def f(x): return x**2 + 2*x + 1
def df(x): return 2*x + 2

x = 10.0
lr = 0.1
trajectory = [x]
for _ in range(50):
    x = x - lr * df(x)
    trajectory.append(x)
print(f"\n梯度下降: 最优点 x ≈ {x:.6f}")

"""
L4-numpy-vectorization.py
NumPy向量化演示
安装依赖: pip install numpy
"""

import numpy as np
import time

n = 10_000_000
a = np.random.rand(n)
b = np.random.rand(n)

# Python循环（慢）
start = time.time()
result_loop = sum(a[i] * b[i] for i in range(n))
loop_time = time.time() - start

# NumPy向量化（快）
start = time.time()
result_vec = np.dot(a, b)
vec_time = time.time() - start

print(f"循环耗时: {loop_time:.4f}s")
print(f"向量化耗时: {vec_time:.4f}s")
print(f"加速比: {loop_time/vec_time:.1f}x")
"""
L4-oop-neural-network.py
面向对象神经网络
安装依赖: pip install numpy
"""

import numpy as np

class Layer:
    def forward(self, x): raise NotImplementedError
    def backward(self, grad): raise NotImplementedError

class Dense(Layer):
    def __init__(self, input_dim, output_dim):
        self.W = np.random.randn(input_dim, output_dim) * np.sqrt(2.0/input_dim)
        self.b = np.zeros((1, output_dim))
        self.x = None

    def forward(self, x):
        self.x = x
        return x @ self.W + self.b

    def backward(self, grad):
        self.grad_W = self.x.T @ grad
        self.grad_b = np.sum(grad, axis=0, keepdims=True)
        return grad @ self.W.T

class ReLU(Layer):
    def __init__(self): self.mask = None
    def forward(self, x):
        self.mask = (x > 0)
        return np.maximum(0, x)
    def backward(self, grad):
        return grad * self.mask

# 演示
net = Dense(4, 8)
relu = ReLU()
x = np.random.randn(32, 4)
h = relu.forward(net.forward(x))
print(f"输入形状: {x.shape}")
print(f"隐层输出形状: {h.shape}")

"""
L5-bias-variance.py
安装依赖: pip install numpy matplotlib scikit-learn
"""

import numpy as np
import matplotlib.pyplot as plt
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import PolynomialFeatures
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error

np.random.seed(42)
X = np.linspace(0, 4, 50).reshape(-1, 1)
y = np.sin(X.flatten()) + np.random.normal(0, 0.3, 50)

X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3)

# 演示不同复杂度模型
degrees = [1, 10, 30]
for degree in degrees:
    poly = PolynomialFeatures(degree=degree)
    X_train_poly = poly.fit_transform(X_train)
    X_test_poly = poly.transform(X_test)

    model = LinearRegression()
    model.fit(X_train_poly, y_train)

    train_mse = mean_squared_error(y_train, model.predict(X_train_poly))
    test_mse = mean_squared_error(y_test, model.predict(X_test_poly))

    print(f"degree={degree}: 训练MSE={train_mse:.4f}, 测试MSE={test_mse:.4f}")
