## 2. 技术架构

```
用户输入 → 向量化 → 课程知识库检索 → LLM 生成 → 返回答案
```

### 技术栈

| 组件 | 技术 |
|------|------|
| 前端 | React + Chat UI |
| 向量化 | OpenAI Embedding |
| LLM | GPT-4 / DeepSeek |
| 知识库 | 课程内容向量存储 |
