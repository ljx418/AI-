## AI 助教技术方案

### 架构

```
用户输入 → 向量化 → 课程知识库检索 → LLM 生成 → 返回答案
```

### 技术栈

| 组件 | 技术 |
|------|------|
| 前端 | React + Chat UI |
| 向量化 | OpenAI Embedding |
| 知识库 | 课程内容向量化存储 |
| LLM | OpenAI GPT-4 / DeepSeek |

### 数据准备

1. 提取 L01-L35 全部课程文本
2. 分块 (chunk=500, overlap=50)
3. 生成向量嵌入
4. 存储到向量数据库 (本地)

### 功能模块

| 功能 | 输入 | 输出 |
|------|------|------|
| RAG 问答 | 用户问题 | 相关课程答案 |
| 代码辅导 | 代码问题 | 代码解释/建议 |
| 学习建议 | 用户目标 | 学习路径推荐 |

## 视频制作计划

### 制作流程

```
选题 → 脚本 → 录制 → 剪辑 → 上传 → 嵌入
```

### 每课视频内容

| 时长 | 内容 |
|------|------|
| 1min | 课程介绍 |
| 4min | 核心知识点讲解 |
| 3min | 代码演示 |
| 1min | 总结/作业 |

### 设备需求

- 麦克风 (USB)
- 摄像头 (或屏幕录制)
- 剪辑软件 (剪映/Final Cut)
