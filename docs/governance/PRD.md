## 4. 资源模式 (无 token 硬限 + 自治)

### 4.1 自治原则

按用户 2026-06-23 brainstorm 决策:
- **不设 token 硬限**: W3-W8 阶段, Claude 自治资源, 不在阶段中途要求"省 token"
- **子 agent 自由**: 写课程 / 写工具 / 抓语雀 全部可 sub-agent 并行
- **质量 > 速度**: 单课 < 8K 字 不算合格 (W3 后硬指标)

### 4.2 软限参考 (W 阶段参考, 不强制)

| 阶段 | 课程单 agent | 工具 sub-agent | 抓语雀 sub-agent | 治理文档主 agent |
|------|------------|---------------|----------------|----------------|
| W3 | 15 (T1+T2) | 2 (基础+RAG) | 1 (287 滚动) | 1 (主控) |
| W4 | 13 (T4+T6) | 2 (微调+Agent) | 0 (续抓) | 1 |
| W5 | 13 (T3+T5) | 2 (RAG+DeepSeek) | 0 | 1 |
| W6 | 13 (T7+T8) | 2 (项目+行业) | 0 | 1 |
| W7 | 13 (T9) | 2 (公司+融合) | 0 | 1 |
| W8 | 5 (查漏) | 1 (全工具 v1.0) | 0 | 1 (验收) |

### 4.3 抓语雀策略 (287 URL 全量)

按用户 2026-06-17 规则 1: **语雀原文抓取先于开发**。W3 启动前必须先抓完对应主题。

```bash
# Phase 1: 滚动收集所有 URL
node scripts/fetch_yuque_urls_only.mjs
# 目标: docs/yuque_raw/yuque_urls.json >= 287 URL

# Phase 2: 按 9 主题批量抓正文
node scripts/fetch_yuque_full.mjs --limit 287
# 目标: docs/yuque_raw/pending_classify/000_xxx.md ... 287_xxx.md

# Phase 3: 按 9 主题分类
node scripts/classify_yuque.mjs
# 目标: docs/yuque_raw/by_topic/T1_基础/ ... T9_公司落地/
```

**抗反爬**:
- 每轮 sleep 800ms (上限 50 轮)
- 复用 `chrome-yuque-data` user-data-dir 保持 cookies
- 断点续抓: `fetch_yuque_full.mjs` 跳过已落盘 doc
- 失败 doc 写 `docs/yuque_raw/yuque_failed.json`, 人工 review

### 4.4 拆解重写 (非搬运)

每个语雀 doc 抓完后, **不直接搬运**, 走以下流程:
1. 解析 9 主题归一化 (按 §2 主题 ID)
2. 拆解为 1-3 课大纲 (每课 8K 字, 4 代码示例, 2 图)
3. 跟 AI 教案已有内容 (L01-L63) 做交叉引用, 避免重复
4. 写课 → 验收 (字数 + 工具对接 + 链接) → 落盘 lessons.json
5. 写 W 阶段 ACCEPTANCE 报告

## 5. 双轨道 (课程对标 + 工具独立)

按用户 2026-06-23 §6 brainstorm 决策: **双轨道并行**, 互不阻塞。

### 5.1 轨道 1: 课程对标 (T1-T9 9 主题)

- **目标**: 9 主题 100% 覆盖 / 200 课 / 1.06M 字
- **节奏**: W3-W8 月均 22 课 + 180K 字
- **主 agent**: 课程策划 (W3-W8 每月 1 个, 跟 governance 文档主控协调)
- **sub-agent**: 写课 (W 阶段 1 个 sub-agent 写 1 主题, 1-2 周)
- **验收**: W 阶段末 `M11_W*_ACCEPTANCE.md` 必含: 字数 / 工具对接 / PRD 规格 / 链接健康

### 5.2 轨道 2: 工具独立增强 (9 工具 → v1.0)

- **目标**: 9 工具 v0.1 → v1.0, 各自功能完整, 形式超越
- **节奏**: 跟课程并行, 不绑定 W 阶段
- **9 工具 v1.0 路线图**:

| Search | Fuse.js 全文 | + 跨课搜索 + 知识图谱 | 导航 |

## 6. 风险与应对

| 风险 | 影响 | 概率 | 应对 |
|------|------|:----:|------|
| 抓语雀 287 URL 不达标 | 课程线阻塞 | 中 | sub-agent 持续修滚动 bug; 接受 80% 偏差 (≥ 230 URL) |

| 主题归一化边界混淆 | 总字数虚高 | 中 | 跨主题课在 lessons.json 加 `topics: ['T4', 'T5']` 标记, 验收按唯一主题计 |
