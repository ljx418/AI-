## 🔧 如何使用

### 方式 A: 在 draw.io 中打开
1. 访问 https://app.diagrams.net/
2. 菜单 File → Open from Device → 选择 `gap-drawio-m11w2.xml`
3. 6 张图同时显示在一张大画布 (2400×1600)

### 方式 B: 导出 PNG
1. 在 draw.io 中打开后
2. File → Export as → PNG
3. 命名为 `gap-drawio-m11w2.png`

### 方式 C: 嵌入 Markdown
```markdown
![M11 W2 Gap Analysis](./diagrams/gap-drawio-m11w2.png)
```
