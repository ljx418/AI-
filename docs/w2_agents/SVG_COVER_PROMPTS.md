## 实施细节

### minimax image-01 调用

```bash
# 单张生成示例
curl -X POST "$ANTHROPIC_BASE_URL/v1/images/generations" \
  -H "Authorization: Bearer $ANTHROPIC_AUTH_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "image-01",
    "prompt": "<上方 prompt 文本>",
    "size": "1024x768",
    "n": 1,
    "response_format": "url"
  }' | jq -r '.data[0].url'
```

### 落盘流程

1. 调用 minimax image-01，返回 PNG URL
2. 用 Inkscape / rsvg-convert 转 PNG → SVG
3. 保存到 `public/images/lesson-covers/L49.svg` 等
4. 在 lessons_new.jsx 的 lesson.image 字段写路径

### 替代方案（如 image-01 不可用）

- 用 minimax-text-01 直接生成 SVG 文本（text-only 模型）
- 或用本地 drawio 生成 SVG 草图
- 最低要求：每课 1 张 240×180 SVG，统一配色
